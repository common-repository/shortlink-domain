Customise the domain name used to generate shortlink URLs.

== Description ==

This plugin allows you to specify an alternative domain name to use with the built in URL shortening service.

For this to work it requires the chosen domain to be pointed / parked on the hosting account used for your blog.

== Installation ==

To enable your custom domain:

1. Point / park the custom shortlink domain on the hosting account used for your blog.
1. Upload the `shortlink-domain` folder to the `/wp-content/plugins/` directory or install the plugin via wp-admin by searching for "Custom Shortlink Domain".
1. Activate the plugin through the 'Plugins' menu in WordPress if it has not already been activated.
1. Add your custom domain to the "Custom Domain" field on the `Settings > Permalinks` page and save your changes.
